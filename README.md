# Face Expression Music Recommendation 🎵😊😢😲

A smart web application that detects the user's face expression and plays songs based on mood.  
For example:

- 😊 Happy Face → Happy Songs  
- 😢 Sad Face → Sad Songs  
- 😲 Surprised Face → Surprise Mood Songs  
- 😐 Neutral Face → Calm Songs  

This project provides a fun and interactive music experience using facial emotion recognition.

---

## 🚀 Features

- 🔐 User Signup & Login Authentication  
- 🎭 Face Expression Detection  
- 🎵 Mood Based Song Recommendation  
- ☁️ Cloud Storage using ImageKit  
- ⚡ Fast Data Handling with Redis  
- 📱 Responsive UI Design  
- 🌐 Frontend & Backend Deployed on Render  

---

## 🛠️ Tech Stack

### Frontend
- React JS
- SCSS
- Axios

### Backend
- Node.js
- Express.js

### Database
- MongoDB

### Other Tools & Services
- Redis
- Multer
- ImageKit
- JWT Authentication

---

## 📌 How It Works

1. User creates account and logs in  
2. Camera detects facial expression  
3. System identifies mood  
4. Songs are fetched based on detected emotion  


---

## 🔒 Authentication

Only registered users can access the platform.  
Users must signup/login before using the application.
